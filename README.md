# Mobile Price Prediction System

## Overview

This project predicts the price category of a mobile phone using machine learning techniques. The model is trained on mobile specifications such as battery power, RAM, internal memory, and other features.

## Features

* Data Analysis (EDA)
* Machine Learning Model Training
* Mobile Price Category Prediction
* MLOps Workflow
* Model Tracking with MLflow

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* MLflow
* FastAPI
* Docker

## Project Structure

* Dataset
* EDA Notebook
* Trained Model
* API Service
